Automated Cooling System

Indusoft Web Studio

Fecha: 2013

Puedes verlo aqui: click

This project was created by students Julián David Orozco Colorado and Felipe Chujfi Bonilla, and supervised by teacher and engineer William Prado Martínez and Julian Eduardo Granados Piravan. The main objective was to design an automated trainer that refrigerates (cools) using compressed vapors R134A that allowed them to measure the temperature between different stages of the cooling systems’ cycle. “We made the instrumentation of each one of the operating points of the system, using temperature sensors, an electronic circuit to the adaptation of the signals and a data acquisition phase. Finally we established a link from a PLC Thinget to Indusoft Web Studio Educational v7.1 with Modbus RTU communication to visualize the whole process” - Julián David Orozco Colorado and Felipe Chujfi Bonilla.
